# OpenPlazma 0.1-alpha.0 Release Notes Draft

Status: draft

This file is the source draft for a future GitHub Release. Do not create the release from this PR.

## Summary

OpenPlazma `0.1-alpha.0` is the first public alpha for a local-first, `STATIC_FIXTURE`-only learning workflow. It connects the Lab, Workbench, StudyFlow, local RunStore, Observatory, and Compare into one safe Read the Signal path.

The alpha is meant for early public review of the workflow, language, documentation, and safety boundaries. It is not a validated simulator, not a reactor design tool, and not a real hardware control system.

## Public Demo

```text
https://mishima-computing.github.io/openplazma/
```

The public demo is static and browser-only. It uses `STATIC_FIXTURE` data only and does not inspect a user's local `.openplazma/` directory.

## First Guided Path

The first guided path is Read the Signal:

1. Enter the Lab.
2. Inspect a `STATIC_FIXTURE` Signal.
3. Write one observation.
4. Write one hypothesis.
5. Run the local guided StudyFlow.
6. Inspect RunStore output.
7. Open the local Observatory.
8. Compare two local Runs.

The local guided workflow is:

```sh
python scripts/run-guided-study-flow.py --run-store .openplazma --clean
```

## Local Technical Workflow

Run the local smoke workflow from the repository root:

```sh
rm -rf .openplazma
python scripts/run-guided-study-flow.py --run-store .openplazma --clean
```

Expected output:

- two local Runs under `.openplazma/runs/...`
- local Observatory index at `.openplazma/observatory/index.html`
- local Compare page under `.openplazma/observatory/compare/...`

Generated `.openplazma/` output is ignored by git and must not be committed.

## Safety And Scientific Scope

OpenPlazma `0.1-alpha.0` is a safe learning workflow for observing, recording, and comparing `STATIC_FIXTURE` signal-like data.

Current safe targets are:

- `static_fixture`
- `local_run_store`

Default public/demo capabilities keep:

```json
{
  "readData": true,
  "writeArtifacts": true,
  "runSimulation": false,
  "submitComputeJob": false,
  "readFacilityTelemetry": false,
  "controlFacility": false
}
```

The project does not provide instructions for high-voltage, vacuum, laser, radiation, hazardous materials, physical plasma hardware, or real facility operation.
