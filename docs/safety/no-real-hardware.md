# No Real Hardware

OpenPlazma is not a real hardware control system.

Do not use this repository to control, automate, design, tune, or operate physical equipment. The project must not include instructions for high voltage, vacuum systems, lasers, radiation sources, hazardous materials, or hazardous experiments.

Allowed scope:

- Static fixture records.
- Data-contract validation.
- Educational visualization.
- Notebook and UI workflows that consume safe local data.

Disallowed scope:

- Live device control.
- Operational procedures for laboratory equipment.
- Hazardous experiment setup or troubleshooting.
- Safety-critical monitoring or interlocks.
- Claims that OpenPlazma validates a physical fusion process.
