# Read The Signal Tutorial

Welcome to the first OpenPlazma Tutorial mission.

Mission: inspect a `STATIC_FIXTURE` signal, write a Logbook observation and hypothesis, run the local guided StudyFlow, inspect RunStore output, open the Observatory, and Compare two Runs.

Expected time:

- Browser tour: 10 minutes.
- Local technical path: 30 minutes.

## What You Will Learn

- Inspect a `STATIC_FIXTURE` Signal.
- Separate observation from hypothesis.
- Understand an ExperimentContext.
- Create local Runs.
- Inspect RunStore output.
- Export Observatory.
- Compare two Runs.
