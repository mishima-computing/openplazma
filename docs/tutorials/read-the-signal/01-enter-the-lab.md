# 01 Enter The Lab

Open the public Lab:

```text
https://mishima-computing.github.io/openplazma/
```

## Control Room Check

Confirm:

- The Lab loads.
- A sample shot is visible.
- A Signal chart is visible.
- The demo scope is `STATIC_FIXTURE`-only.

## Open The Notebook

If the button is available, choose `Open Experiment Notebook`.

The Workbench opens in the browser. The browser tour does not write to `.openplazma/` and does not inspect a local RunStore.

## Mission Boundary

The browser tour uses `STATIC_FIXTURE` data only.
It does not fetch external fusion data.
It does not control hardware.
