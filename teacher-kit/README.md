# Teacher And Workshop Pack

This directory contains workshop materials for using OpenPlazma in a short safe learning session.

The first Workshop Pack is:

- Read the Signal, 45 minutes

These materials are intended for:

- teachers
- science communicators
- workshop facilitators
- student groups
- developer reviewers

## What Participants Do

Participants use the existing Read the Signal guided StudyFlow. Depending on the session setup, they may:

1. Open the public Lab.
2. Inspect a `STATIC_FIXTURE` signal.
3. Write one observation and one hypothesis.
4. Optionally run the local Notebook and RunStore workflow.
5. Optionally inspect local Observatory and Observatory Compare output.

The browser-only path can be used without local setup. The local technical path is optional and uses repository scripts.

## Limitations

- `STATIC_FIXTURE` data only.
- No grading or scoring.
- No public data ingestion.
- No AI assist.
- No real hardware instructions.
- Not a validated fusion simulator.
- Not a reactor design tool.
- Not a real hardware control system.
