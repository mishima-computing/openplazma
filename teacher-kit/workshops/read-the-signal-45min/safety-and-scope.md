# Safety And Scope

## What OpenPlazma Is

OpenPlazma is a local-first experiment and learning system for safe signal inspection workflows.

The public demo uses `STATIC_FIXTURE` data only.

## What OpenPlazma Is Not

- Not a validated fusion simulator.
- Not a reactor design tool.
- Not a real hardware control system.
- No public data ingestion.
- Not a hosted Observatory service.
- Not AI assist.

## Safety Boundaries

This Workshop Pack does not include:

- No real hardware instructions
- hazardous experiment procedures
- facility operation steps
- external machine, reactor, or device integration

## Answering Safety-Sensitive Questions

Use short boundary language:

- "That is outside the public demo scope."
- "This workshop uses `STATIC_FIXTURE` data only."
- "OpenPlazma does not provide real hardware instructions."
- "We can discuss the observation workflow without operational procedures."

If a question asks for hazardous procedures, redirect to the safe learning goal: observe the fixture signal, write an observation, write a hypothesis, and inspect local records.
