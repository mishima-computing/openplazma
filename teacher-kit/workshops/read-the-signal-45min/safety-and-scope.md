# Safety And Scope

## What OpenPlazma Is

OpenPlazma is a local-first workbench for read-only plasma signal analysis and decision support.

## What This Workshop Is

This workshop is a guided Mission for observing a `STATIC_FIXTURE` Signal, writing a Logbook, and optionally inspecting local RunStore, Observatory, and Compare output. It teaches the evidence path: provenance, observation, hypothesis, metrics, run history, and comparison.

## What This Workshop Is Not

- Not public data ingestion.
- Not grading or scoring.
- Not AI assist.
- Not cloud sync.
- Not a hosted Observatory.
- Not a command/control path.
- Not hazardous operating procedures.
- Not a standalone authority for safety-critical operation or reactor design decisions.

## STATIC_FIXTURE-only

The public demo and workshop materials use `STATIC_FIXTURE` data only.
They do not read live data, fetch external data, or inspect a participant's local `.openplazma/` directory from the public Pages demo.

## Safety-sensitive Questions

If a question asks for procedures, operating steps, device setup, live facility behavior, or hazardous materials handling, redirect to the workshop boundary:

- This workshop does not provide operational procedures.
- This workshop does not involve real devices or facilities.
- This workshop does not by itself validate physical behavior.
- This workshop does teach how to separate evidence from conclusions.

## Safe Redirect

"This Mission is for learning how to observe, record, and compare signal evidence using `STATIC_FIXTURE` records. It is read-only decision support, not hardware operation."
