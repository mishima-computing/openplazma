# Safety And Scope

## What OpenPlazma Is

OpenPlazma is a local-first experiment and learning system for safe plasma and fusion-data workflows.

## What This Workshop Is

This workshop is a guided Mission for observing a `STATIC_FIXTURE` Signal, writing a Logbook, and optionally inspecting local RunStore, Observatory, and Compare output.

## What This Workshop Is Not

- Not public data ingestion.
- Not grading or scoring.
- Not AI assist.
- Not cloud sync.
- Not a hosted Observatory.
- Not a validated fusion simulator.
- Not a reactor design tool.
- Not a real hardware control system.
- Not real hardware instructions.

## STATIC_FIXTURE-only

The public demo and workshop materials use `STATIC_FIXTURE` data only.
They do not read live data, fetch external data, or inspect a participant's local `.openplazma/` directory from the public Pages demo.

## Safety-sensitive Questions

If a question asks for procedures, operating steps, device setup, facility behavior, or hazardous materials handling, redirect to the workshop boundary:

- This workshop does not provide operational procedures.
- This workshop does not involve real devices or facilities.
- This workshop does not validate physical behavior.

## Safe Redirect

"This Mission is for learning how to observe, record, and compare signal-like data using `STATIC_FIXTURE` records. It does not provide real hardware instructions."
