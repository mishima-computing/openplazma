# Read the Signal: 45-minute Workshop

Duration: 45 minutes.

This Workshop Pack supports the mission-style [Read the Signal Tutorial](../../../docs/tutorials/read-the-signal/README.md) and the local [Guided StudyFlow](../../../docs/guided-study-flow.md).

The workshop frames OpenPlazma as read-only signal analysis and decision support: participants inspect evidence, write assumptions, log a run, and compare outputs without controlling equipment.

## Format

Browser-only path:

- Uses the public Lab.
- Requires only a browser.
- Stops after observation, hypothesis, and Debrief.

Optional local technical path:

- Uses the local repository.
- Runs the guided StudyFlow.
- Creates local Runs.
- Opens local Observatory and Compare output.

## Recommended Audience

- First-time OpenPlazma users.
- Teachers and science communicators preparing a guided session.
- Student groups learning observation and hypothesis separation.
- Developer reviewers checking the local workflow.

## Prerequisites

Browser-only path:

- A browser.
- Access to the public demo.

Local technical path:

- Git.
- Node/Corepack.
- Python.
- The OpenPlazma repository.

## Workshop Boundary

This workshop uses `STATIC_FIXTURE` data only.
It does not provide grading, scoring, public data ingestion, cloud sync, hosted Observatory, AI assist, command/control actions, hazardous procedures, or standalone safety-critical design decisions.
